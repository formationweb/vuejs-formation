Chaque carte - ou vignette - est un sous composant du composant Users ça représente un seul utilisateur 

A partir de la propriétés “users” - ajoutée ensemble - dans le composant Users:


1. Créez le composant UserCard
2. Affichez autant de fois le composant UserCard qu'il y a d'utilisateurs dans le tableau
3. Le composant UserCard à son propre template. Vous devez afficher le nom et l’adresse email de la personne

4. (extra) Créez l’interface “User” en TypeScript
